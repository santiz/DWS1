Element Definitions

Chapter 2 Text

Headings (<h1>,<h2>,<h3>, etc)
These Heading Tags, describe structure within a document. From, most important, to least. Also can be used to describe the order in which things are read. <h1> being the main heading, and followed by <h2> and so on.
Paragraph (<p>)
Is the symbol used in order to tell the browser to incase that specific group of text within the “<p></p>” tags, into a paragraph, and anything within, is a paragraph indeed.
Bold & Italic (<b> and <i>)
The <b> and <i> tags are used to create text that is bold like this or text that is italicized, like this. This provides the user whom is reading it with the true meaning of what the specific word or paragraph is describing in its true sense.
Superscript <sup> and Subscript <sub>
<sup> is the element used to describe symbols such as: " th " when describing suffix or a mathematical number raised to a certain power. While the <sub> tag defines a subscript, meaning something such as a footnote or a chemical formula like H2O.
Line Breaks and Horizontal Rules
Line breaks can be used when one, wants to break a line from a paragraph or a group of text, using the <br /> tag. One can also use a certain element, to add a line break using the <hr /> tag, these especially come in handy when you want to break text into a new topic for example.
Strong <strong> and Emphasis <em>
These type of tags provide a feel to the way the sentence or the group of text is read. It does a similar job to Italicized text or Bold text.
Quotations
If you wanted to quote a really long quote, you would use the <blockquote> tag to emphasize that to the user.
Abbreviations and Acronyms
The abbreviations <abbr> tag, can be used in the document, to quickly abbreviate things like Professor, Boulevard, or other words that tend to get abbreviated. Whereas the <acronym> tag can be used to abbreviated departments such as NASA, DEA, FBI, and others alike. Which would show the full meaning on mouse roll over: FBI.
Citations and Definitions
A <cite> tag would italicize the given words into a citation, which you would use to reference a book or a given address of a specific document one was referring to. Now if you were defining a specific term within a document, you can always assign a <dfn> tag, which is used to define the instance of that object.
Author Details or the <address> tag
This tag can specifically be used to identify personal details of the specific author. Doesn't necessarily mean to give an address within this tag, but can be used for many different options.
Changes to Content
The <ins> & <del> can be used to insert and delete previous recorded text within the document. The <ins> tag simply inserts a new word after whatever word you added the <del> tag. The <del> will put a line across the specific word, while the <ins> underlines it.

Chapter 3 List

Ordered and Unordered List
Both type of lists require their own individual elements. <ol> for ordered and <ul> for unordered list. The ordered list tag, will put everything in a specific number count, while the unordered list will make a bullet out of it. In both settings, you need your <li> tag to create that specific instance within that list as well.
Definition List
A definition list is in effect at the moment, within this very document. It's triggered by your <dl> tag, which creates the body for the terms. Then, it also consists of two elements; <dt>, which is the term being defined and <dd>, which defines the term.
Nested List use the same concepts as the unordered list (same elements), except they are within the same list. so you would add an additional <li> tag within an original <li> tag to create a list that is nested.
